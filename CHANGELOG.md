# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-12-09

### Added
- Initial release of active_dataflow-connector-sink-json_rpc
- JsonRpcSink class for sending data via JSON-RPC client
- Single record and batch write operations
- Automatic buffering with configurable batch size
- Thread-safe buffer management
- Connection testing and health check capabilities
- Graceful error handling with detailed logging
- JSON serialization/deserialization support
- RSpec test suite with comprehensive coverage
- Cucumber feature tests for behavior verification
- Complete documentation and usage examples
